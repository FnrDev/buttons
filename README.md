# Discord Buttons


## An easy bot to create discord buttons easily.

---
### Note: Node.js 16.6.0 or newer is required.
---

## Installation

```sh
npm install
```
---

### You need to rename example.env file to `.env` and fill the info.

- .env
```sh
TOKEN=YOUR_DISCORD_BOT_TOKEN
BOTID=BOT_ID
SERVERID=SERVER_ID
OWNERS=["596227913209217024"]
```
